# GestãoEscolar
 
Disciplina: Desenvolvimento de Aplicações Web II

## Objetivo

O GestãoEscolar é uma aplicação web desenvolvida para auxiliar no gerenciamento acadêmico de turmas, alunos e notas escolares, permitindo o controle de médias, conceitos e recuperação.

## Funcionalidades

- Cadastro de turmas.
- Cadastro de alunos vinculados às turmas.
- Lançamento de notas bimestrais.
- Cálculo automático de média.
- Definição automática de conceitos: A, B, C ou D.
- Exibição de mensagens conforme o desempenho do aluno.
- Sistema de recuperação para alunos com conceito C.
- Fechamento de turmas para impedir alterações futuras.
- Interface utilizando Bootstrap.

## Tecnologias Utilizadas

- PHP
- Laravel
- Bootstrap
- SQLite
- HTML
- CSS

## Finalidade Acadêmica

Trabalho prático desenvolvido para fins de aprendizado em desenvolvimento web utilizando Laravel, banco de dados relacional e regras de negócio no tema Notas de Alunos.
